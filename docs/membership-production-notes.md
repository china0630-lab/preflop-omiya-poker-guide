# 会員制サイト化メモ

## 現在の状態

現時点では、会員登録とログインはフロントエンドだけの簡易実装です。

- ユーザー情報: `localStorage`
- 招待コード: `script.js` 内の `INVITE_CODES`
- 会員限定制御: ブラウザ上の表示制御

これはベース確認用で、本番のセキュリティ用途には不十分です。

## 本番で必要なもの

- サーバー側のユーザー管理
- パスワードの安全な保存
- 招待コードのサーバー側検証
- 会員限定コンテンツのサーバー側保護
- パスワード再設定
- 管理者による招待コード発行

## 推奨構成

最短で進めるなら、Netlifyと相性の良い次の構成が現実的です。

```text
Netlify Hosting
Supabase Auth
Supabase Database
```

記事は最初は `content/articles.js` 管理で始め、会員数が増えたらCMSやDBへ移行できます。
